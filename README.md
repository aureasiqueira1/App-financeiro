## ⚙️ Como executar

Clone o projeto em sua máquina
```bash
  git@github.com:aureasiqueira1/App-financeiro.git
```

abra o projeto
```bash
  cd App-financeiro
```

Instale as dependencias para o frontend(com yarn or npm)
```bash
  yarn install
```

Inicie o projeto
```bash
  yarn start
```
